# Chinthapally Aravind — Personal Portfolio

Task 1 deliverable for the ApexPlanet Full Stack Web Development Internship
(PHP & MySQL track): a personal portfolio site built with plain HTML, CSS,
and JavaScript, and deployed via GitHub Pages.

## 📁 Project structure
```
portfolio/
├── index.html      # Page structure and content
├── style.css       # Design system, layout, responsive rules
├── script.js       # Typing effect, mobile nav, scroll reveal
└── README.md
```
